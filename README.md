# FokusFlow

Pomodoro-style focus timer with daily and weekly task planning.

## Features
- Focus/Break/Long Break timer with auto-switch option
- Daily task list with progress, reorder, and completion history
- Weekly planner with copy/paste for repeating tasks
- Live clock (WIB)
- Sound notification when session ends
- English and Indonesian UI (toggle in header)

## Notes
- Data is stored in `localStorage` (browser only).
- Notification sound file is in `assets/school-bell.mp3`.
- Weekly planner is in `weekly.html`.

## Pages
- `index.html`: main timer + daily tasks
- `weekly.html`: weekly tasks
